# ETL Pipeline with Apache Spark

## Описание работы

В рамках данной лабораторной работы реализован ETL-пайплайн с использованием Apache Spark для обработки данных, их нормализации и построения аналитических витрин.

Исходные данные представлены в виде CSV-файлов и содержат информацию о продажах, клиентах, продуктах, магазинах и поставщиках.

---

## Цель работы

- Реализовать ETL-процесс
- Построить модель данных "звезда" (Star Schema)
- Создать аналитические витрины
- Использовать ClickHouse как OLAP-хранилище

---

## 🏗 Архитектура решения

CSV → PostgreSQL (RAW) → Spark → Star Schema → Spark → ClickHouse (Data Marts)


---

## 🗄 Используемые технологии

- Apache Spark — обработка данных
- PostgreSQL — хранилище исходных и нормализованных данных
- ClickHouse — аналитические витрины
- Docker — окружение

---

## 📊 Анализ исходных данных

Исходные данные представляют собой денормализованную таблицу, содержащую:

- клиентов
- продукты
- магазины
- поставщиков
- продажи

Проблемы:
- дублирование данных
- отсутствие структуры
- невозможность эффективной аналитики

---

## Модель данных (Star Schema)

Для решения проблемы была построена модель "звезда".

### Факт таблица

- fact_sales — содержит данные о продажах

### Измерения

- dim_customer — клиенты
- dim_product — продукты
- dim_seller — продавцы
- dim_store — магазины
- dim_supplier — поставщики
- dim_time — временное измерение

---

## ⚙️ ETL процесс

### 1. Extract
Данные загружаются из CSV в PostgreSQL.

### 2. Transform
С помощью Spark выполняется:
- очистка данных
- удаление дубликатов
- формирование измерений
- генерация surrogate keys
- построение факт таблицы

### 3. Load
Данные записываются:
- в PostgreSQL (модель "звезда")
- в ClickHouse (витрины)

---

## Витрины данных

### 1. Витрина продуктов
- Топ-10 самых продаваемых продуктов
- Выручка по категориям
- Средний рейтинг
- Количество отзывов

### 2. Витрина клиентов
- Топ-10 клиентов по сумме покупок
- Распределение клиентов по странам
- Средний чек

### 3. Витрина времени
- Месячные продажи
- Годовые продажи
- Средний чек по месяцам

### 4. Витрина магазинов
- Топ-5 магазинов
- Продажи по городам и странам
- Средний чек

### 5. Витрина поставщиков
- Топ-5 поставщиков
- Средняя цена товаров
- География поставщиков

### 6. Витрина качества
- Продукты с лучшим и худшим рейтингом
- Зависимость продаж от качества
- Количество отзывов
